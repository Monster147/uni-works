# Trabalho Prático 2
Desenvolvimento de um API

`npm run testServices` - Testa as funções que se encontra no ficheiro foccacia-services.

`npm run testDataMem` - Testa as funções que se encontra no ficheiro foccacia-data-mem.

`npm run testTeamsData` - Testa as funções que se encontra no ficheiro fapi-teams-data.

O ficheiro `.env` deve conter as variáveis `API_KEY` e `PORT`, exemplo:

```
API_KEY = 458n6q9g18q014qr3lm74b5h6q147480
PORT = 8080
```
Utilizamos este método para não expor keys privadas.

Para além da criação do ficheiro `.env`, também será necessário por a `API_KEY` no ficheiro `.http`
